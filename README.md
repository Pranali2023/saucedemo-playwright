## Overview
This project contains an automated test suite built using Playwright (JavaScript)
to validate a successful checkout flow on the Sauce Labs demo website.

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Test Coverage
- User login
- Select 3 random products
- Add items to cart
- Complete checkout
- Verify successful order confirmation

## Setup
1. Install dependencies
   npm install

2. Install Playwright browsers
   npx playwright install

## Run Tests
npx playwright test

## View Report
npx playwright show-report
